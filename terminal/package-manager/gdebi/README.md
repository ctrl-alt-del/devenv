gdebi
=====
[**gdebi**](https://launchpad.net/gdebi) is a tool for installing `deb` files.

* [**Ubuntu**](#gdebi-on-ubuntu)
* [**Fedora**](#gdebi-on-fedora)

## gdebi on Ubuntu
```sh
sudo apt-get install gdebi
sudo apt-get update
```
