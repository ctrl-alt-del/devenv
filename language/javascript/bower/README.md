Bower
=====
[**Bower**](http://bower.io) is a package manager for the web.  You need [npm](#npm) in order to install `bower`.

* [**Mac**](#bower-on-mac)
* [**Ubuntu**](#bower-on-ubuntu)
* [**Fedora**](#bower-on-fedora)

### Bower on Mac
```sh
npm install -g bower
```
