Openswan
========
[Openswan]() is an open source IPsec implementation for Linux.

* [**Ubuntu**](#openswan-on-ubuntu)
* [**Fedora**](#openswan-on-fedora)

### Openswan on Ubuntu
```sh
sudo apt-get update
sudo apt-get install openswan
```

### Openswan on Fedora
```sh
sudo yum install openswan
```
